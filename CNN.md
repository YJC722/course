## 一、主題

隨著都市的快速發展，人口湧入都市也為現代都市增添了許多交通問題，傳統交通管理方式難以有效解決道路安全與行車效率等問題。隨著近年來深度學習技術的成熟，街景影像分析成為智慧交通不可或缺的一部分，也使智慧交通與自動駕駛技術逐漸成為解決交通壅塞與降低事故發生機率的重要方向。

因此，本專題旨在比較不同深度學習平台與卷積神經網路架構於街景影像辨識上的差異，研究對象為 MATLAB 平台之 AlexNet 與 Google 所提供之 Teachable Machine 平台影像分類模型，探討其在街景圖片分類準確率及訓練效率的差異。

研究過程的第一步是蒐集並整理街景影像資料，本專題以國家為分類收集街景，收集了台灣、巴西和美國三個國家各 50 多張照片，並依國家進行標註。第一個方法是使用 MATLAB 的 AlexNet 進行街景影像辨識，透過調整訓練參數及替換訓練資料，使訓練正確率提升。另一方面，在 Google 的 Teachable Machine 平台中，可以直接將影像資料上傳，並依照研究需求進行標籤分類，平台就會自動進行模型訓練，與 MATLAB 相比，這個平台不需要設定複雜的訓練參數，即可得到辨識影像的模型，訓練完成後，也可以直接在平台上測試模型效能，並匯出模型。

此外，為確保實驗結果可比較，兩種方法皆採用相同的訓練資料，本次實驗的評估指標為分類的準確率，最後根據實驗結果分析 AlexNet 與 Teachable Machine 平台於街景影像辨識上的特性。

## 五、實驗結果討論

訓練結果顯示 AlexNet 的整體分類準確率約為 70%~79%，對三個國家的辨識能力較佳且相對均衡。而在 Google 的 Teachable Machine 平台上，模型對台灣街景辨識達到 100% 的準確率，但對美國與巴西街景的辨識表現不佳，準確率僅約 40%。我推測可能的原因是因為平台使用的預設模型為較初階的模型，或者訓練時長較短，導致辨識率比起 AlexNet 略顯遜色 [cite: 601]。

除此之外，在初次訓練時我收集的圖片皆為都市化程度較高的地區或者公路的圖片，因此導致三個國家的圖片特徵相似，使訓練的準確率只有將近 30%~40%，後來我才把訓練圖片改成特色較明顯的圖片，像是台灣的圖片大多包含騎樓街景或者小巷，美國大多包含別墅或者較寬廣的路，巴西的圖片大多包含熱帶植被或者土鋪的路，改完之後才使訓練準確率提升到 70% 左右 [cite: 602]。

綜合比較，在台灣街景方面 MATLAB 的 AlexNet 辨識準確率雖然不及 Teachable Machine 辨識準確率，但在三個國家整體分類上表現較穩定；Teachable Machine 雖然可以方便且快速的建立模型，但對相似或較複雜的特徵辨識率偏低 [cite: 603]。

## 六、心得感想

本次專題主要目標為研究與實作圖像辨識模型，並比較 MATLAB 與 Google Teachable Machine 兩個平台的模型表現。在資料準備階段，我收集了台灣、巴西與美國三個國家的街景影像，但是由於拍攝設備的差異、拍攝角度與又或者光線條件不同，使得原始資料本身具有一定的複雜性，像是較先進的地區例如美國紐約大部分資料都是廣角、環景拍攝，使單張圖片會使街景看起來十分扭曲，又或者巴西較偏鄉的地區有些圖片解析度偏低，需要在下載資料時避免使用這些資料 [cite: 605]。

在訓練時，我發現在不同國家收集相似特徵的照片會使訓練率降低，推測原因可能是訓練集只有 50 多張照片，這種較複雜的圖片可能需要更多資料來讓訓練準確率提高，也可能是因為街景圖片本身較複雜，不適合使用 AlexNet 這種較基礎的模型 [cite: 606]。

透過本次專題，我學會了訓練圖像辨識模型的完整流程，像是資料預處理、訓練參數與資料集的數量都是影響訓練結果的關鍵，除了訓練的資料，不同平台的網路架構與訓練流程也對結果影響很大。未來如果要進一步提升準確率，可考慮增加資料量或更複雜的深度網路架構 [cite: 607]。

綜上所述，這次專題不僅讓我熟悉了深度學習影像分類流程，也增強了我篩選資料與訓練模型的能力，為我在圖像辨識相關領域奠定了紮實的基礎 [cite: 608]。
